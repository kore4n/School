◆ addReminder()

void Reminder::addReminder ( )

This function prompts the user to enter a date, time, and message for the new reminder. The reminder is added to the appropriate date in the reminders map and written to the reminders.txt file. The reminder.sh shell script is executed to update the system's cron job scheduler.
